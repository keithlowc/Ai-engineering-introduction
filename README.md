# Keith Low - AI Engineer Freelancer Website

## Overview
This is a professional portfolio website for Keith Low, showcasing his expertise as an AI Engineer and freelancer. The website highlights his capabilities in machine learning, natural language processing, computer vision, and data science.

## Features

### 1. **Responsive Design**
- Mobile-friendly layout that works on all devices
- Smooth scrolling and intuitive navigation
- Modern gradient designs and animations

### 2. **Sections Included**
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Professional background and expertise summary
- **Capabilities Section**: 6 key capabilities with icons and descriptions
  - Machine Learning Models
  - Natural Language Processing
  - Computer Vision
  - Predictive Analytics
  - AI Integration
  - Data Science & Analysis
- **Portfolio Section**: Featured projects showcasing real-world work
- **Contact Section**: Contact information and inquiry form
- **Navigation**: Sticky navbar with smooth navigation

### 3. **Interactive Elements**
- Smooth scroll navigation
- Hover effects on cards and buttons
- Form validation
- Scroll-triggered animations
- Floating card animation in hero section

## Technology Stack
- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **JavaScript**: DOM manipulation and interactivity
- **Google Fonts**: Poppins font family

2. Open `index.html` in your web browser
3. No build process or dependencies required

## Customization

### Updating Contact Information
Edit the contact section in `index.html`:
```html
<a href="mailto:keith@keithlowdev.com">keith@keithlowdev.com</a>
<a href="https://keithlowdev.com">keithlowdev.com</a>
```

### Modifying Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    /* ... other colors ... */
}
```

### Adding New Capabilities
Add new cards to the capabilities grid:
```html
<div class="capability-card">
    <div class="capability-icon">🔧</div>
    <h3>Your Capability</h3>
    <p>Your description here</p>
</div>
```
