# Kenny Low - Personal Website

A professional website for Kenny Low, showcasing his transition from Accenture Tech Consultant to Real Estate Wholesaler.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional design with gradient backgrounds
- **Smooth Animations**: Engaging scroll animations and transitions
- **Contact Form**: Easy way for clients and partners to reach out
- **Multiple Sections**: Hero, About, Expertise, and Contact sections

## Sections

### Hero Section
Introduces Kenny Low with a compelling headline and call-to-action button.

### About Section
Highlights Kenny's background as an Accenture consultant and current role as a real estate wholesaler, with key statistics.

### Expertise Section
Showcases six areas of expertise:
- Real Estate Wholesaling
- Technology Consulting
- Data Analysis
- Partnership Development
- Process Optimization
- Business Strategy

### Contact Section
Includes a contact form, email, phone number, and social media links.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No additional dependencies or build tools required

## Customization

### Colors
The primary color scheme uses:
- Primary Purple: `#667eea`
- Secondary Purple: `#764ba2`

To change colors, update the values in `styles.css`.

### Contact Information
Update the contact details in the Contact section of `index.html`:
- Email: `kenny@kennylow.com`
- Phone: `(555) 123-4567`
- Social Media Links

### Form Handling
Currently, the form logs data to the console. To make it functional:
1. Set up a backend service (Node.js, Python, etc.)
2. Update the form submission handler in `script.js`
3. Integrate with an email service (SendGrid, Mailgun, etc.)
